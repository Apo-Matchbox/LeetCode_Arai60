## 1. 概要

本リポジトリは、コーディング面接対策としてLeetCodeの代表60問をC++で学習・記録するためのものです。

---

## 2. 目的

- 毎日継続的に問題を解く習慣をつける  
- コーディング力と問題解決力を高める  
- コードレビューによるフィードバックで質を高める  

---

## 3. 学習手順

1. 1日1〜3問を選んで解く  
2. READMEに進捗と考察を記録  
3. 必要に応じてコードレビューを依頼  
4. 類題や復習にも取り組む

---

## 4. 教材


基本[コーディング面接対策のために解きたいLeetCode 60問] (https://1kohei1.com/leetcode/)に収録されているLeetCode問題集に沿って学習を進める。

主に以下の60問を使用：
